# Project-Django-Basics-MVT-Views-Templates-
Objective: Understand Django MVT architecture, function-based views, and template rendering.



Create a Django project and perform the following tasks: 



1. MyPortfolio Project / Profiles App

* Create Django project myportfolio and app profiles.

* Display profile via FBV and context.

2. Dynamic Route

* Add /user/<name>/ route to display username dynamically.

3. MyBlog Mini App

* Create Post model with title, content, created_at.

* Build a homepage view listing posts.

4. Post Creation Form

* Implement GET/POST logic to add a new post and redirect to home.

5. Customize Admin Panel

* Show only username and email fields.
